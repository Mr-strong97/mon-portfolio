# MrStrong01 — Portfolio

Portfolio one-page professionnel pour **Isaac Mbuyama (MrStrong01)** — Fullstack Developer, AI Generative Specialist & UX Designer.

**Stack** : React 18 + TypeScript + Tailwind CSS + Lucide React + Vite

## Aperçu

Un portfolio moderne en thème **bleu** (`#2563EB`), 100% responsive, avec :

- Navbar sticky avec scroll-spy et menu mobile animé
- Hero avec effet de frappe (typing effect), stats animées et cartes flottantes
- Section À propos avec carte "Culture Code" et philosophie design
- Section Compétences avec filtres par catégorie et barres de progression animées
- Timeline d'expériences professionnelles
- Formations & certifications
- Grille de projets filtrable (Web / Mobile / IA / Design / Équipe)
- Carousel de témoignages auto-défilant
- Formulaire de contact avec validation temps réel
- Footer complet avec réseaux sociaux

## Démarrage

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build de production
npm run build

# Prévisualiser le build
npm run preview
```

Le site est accessible sur http://localhost:5173

## Structure du projet

```
src/
```
